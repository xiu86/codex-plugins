# 行为准则

## 我们的承诺

我们希望本项目成为尊重、专业、高效的协作空间。贡献者和维护者应以建设性方式沟通，聚焦项目改进。

## 鼓励的行为

- 尊重他人，保持专业。
- 提供可执行的反馈。
- 让讨论聚焦在项目本身。
- 发现错误后及时承认并修正。
- 尊重不同经验水平和上下文背景。

## 不可接受的行为

- 骚扰、威胁或人身攻击。
- 歧视性语言或行为。
- 未经允许公开他人隐私信息。
- 反复干扰项目讨论。
- 维护者合理认定为不适合开源项目协作的其他行为。

## 执行方式

维护者可以删除评论、关闭讨论或限制违反本准则者参与项目。严重或反复违反者可能被永久禁止参与项目空间。

## 反馈方式

如需报告行为问题，请通过 `.codex-plugin/plugin.json` 中列出的维护者联系方式，或代码托管平台提供的私密举报能力处理。
